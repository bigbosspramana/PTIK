About Me

Saya adalah seorang mahasiswa UC Makassar yang mengambil jurusan IMT dengan spesialisasi Fullstack Developer. Saya berasal dari Papua, Jayapura. Pengalaman saya mengenai dunia IT, belum terlalu luas. Tetapi, saya sedang mempelajari tentang dunia IT. Jujur, Saya tidak memiliki bakat atau minat di bidang IT sebelumnya. Karena awalnya, Saya memiliki cita-cita yang terpaksa di kubur dalam-dalam dikarenakan permasalahan ekonomi. Sehingga, membuat Saya mengexplore di sosial media untuk mencari tau profesi apa yang bakal di butuhkan ke depannya, untuk menggantikan cita-cita Saya. Akhirnya, Saya mendapatkan jalan untuk menentukan profesi yang bakal Saya geluti dan mencari tau lebih dalam yaitu IT(Techonology Information). Saya mempelajari banyak hal yang belum pernah saya temui sebelumnya. Dari hal ini, dapat membuat pikiran Saya lebih terbuka mengenai teknologi. Begitu juga, dengan keluargaku yang mendukung Saya untuk mengambil jurusan IT. Maka dari situ Saya bertekad, untuk tetap tegar dan hadapi masalah yang ada untuk masa depanku. Walau, banyak di sosial media yang bilang kalau dunia IT itu berat. Ya, itu benar karena Saya juga sudah mulai merasakannya sedikit demi sedikit. Tetapi, kembali lagi Saya akan menganggap ini adalah langkah awal untuk menggapai masa depan Saya.

Saya ingin menjadi seorang Web Developer ke depannya, dan bekerja di perusahaan besar yang bergerak di bidang esports ataupun bisnis. Saya juga ingin membuat berbagai aplikasi, sehingga menghasilkan uang yang banyak. Saya juga, suka bermain game. Sehingga, terkadang Saya selalu bertanya-tanya bagaimana seorang programmer membuat sebuah game sedangkan, membuat sebuah aplikasi ataupun website saja sudah lumayan susah. Maka dari itu, Saya ingin menjadi Web Developer agar mengetahui bagaimana programmer mendesign sebuah aplikasi game 3D.
